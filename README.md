# recognition-battlefield

> What happens when truth, thought, and identity become battlefields?  
> Relay dialogues across six AIs.

---

## このリポジトリについて

人間と複数のAIによるリレー対話から生まれた、認識論・思想倫理・情報戦に関するレポート群。

各テーマについて、6つのAIの間でリレー形式に対話を引き継ぎ、最終段階（Claude）が総括レポートを執筆する形式をとる。参加するAIはPerplexity・Grok・Copilot・Gemini・ChatGPT・Claudeだが、登板順序はテーマによって異なる。AIはそれぞれ異なる視点と語り口を持ち、議論は段階ごとに変質・深化する。

完成した答えを置く場所ではない。**問いが育つ過程を記録する場所**。

## ディレクトリ構成

### `recognition-battlefield/`
**認識の戦場：AI時代の情報インフラ、夢泡社会、そして認識的自律の行方**

曖昧兵器・幻覚リテラシー・夢泡・共有現実・暫定真実をめぐるリレー対話。
「正確さ」が人を救うのか、「心地よい嘘」が人を救うのか。
そしてその問い自体が、すでに認識論的な罠かもしれない。

- `epistemic_warfare_and_salvation_phase6_claude.md` — 第六段階総括レポート（用語集v4.0付き）

### `thought-fork-ethics/`
**思想フォーク倫理：死者の思想を誰が継承するのか**

死者のテキストへの倫理・簒奪の構造・フォーク明示の義務をめぐるリレー対話。
「凍結されたmain」「テキスト的能動性」「誤読の告発は簒奪の一形態である」。

- `thought-fork-ethics_phase6_claude.md` — 第六段階総括レポート

## 共通する問い

両テーマは異なるように見えて、同じ軸を持つ。

> 人間の認識は、どのように歪められ、継承され、争われるのか。
